# Images de Maisons Container

## Structure des dossiers

Placez vos images de maisons container dans ce dossier avec les noms suivants :

### Pour les modèles de maisons (PageAccueil et PageMaisonContainer)
- `modele-1.jpg` à `modele-9.jpg`

### Pour la galerie (PageGaleries)
- `galerie-1.jpg` à `galerie-12.jpg`

### Pour les projets réalisés (PageProjets)
- `projet-1.jpg` à `projet-8.jpg`

## Formats acceptés
- JPG / JPEG
- PNG
- WebP

## Taille recommandée
- Modèles : 800x600px minimum
- Galerie : 1200x800px minimum
- Projets : 1000x700px minimum

## Comment ajouter vos images

1. Placez vos images dans ce dossier (`/public/images/maisons-container/`)
2. Nommez-les selon la convention ci-dessus
3. Les images seront automatiquement intégrées dans le site
